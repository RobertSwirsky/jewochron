# Jewochron App Icon Design

## Design Concept
A modern, elegant icon combining Jewish symbolism with time/calendar elements.

## Color Scheme
- Primary: Deep Blue (#1A4D7A) - Represents tradition and trust
- Accent: Gold (#DAA520) - Represents Jerusalem and holiness
- Background: White or transparent

## Icon Elements

### Design Option 1: Star of David with Clock
- Six-pointed Star of David in gold
- Clock face integrated in the center
- Hebrew letters ◊ñ (Zayin) for "Zman" (time)
- Circular border in deep blue

### Design Option 2: Hebrew Calendar Scroll
- Stylized Torah scroll with calendar pages
- Moon phases visible
- Hebrew date text
- Gold accents on scroll handles

### Design Option 3: Jerusalem Skyline with Clock (RECOMMENDED)
- Simplified Jerusalem skyline silhouette (like in your app!)
- Dome of the Rock golden dome
- Clock hands pointing to significant time
- Deep blue background with gold highlights

## Quick Implementation Using Free Tools

### Option A: Using Paint 3D (Built into Windows)
1. Open Paint 3D
2. Create 1024x1024 canvas
3. Use 2D shapes to create Star of David
4. Add circle in center
5. Add clock hands
6. Save as PNG

### Option B: Using PowerPoint
1. Create slide with 1:1 aspect ratio
2. Insert shapes (triangles for Star of David)
3. Add circle and clock elements
4. Group all elements
5. Right-click ‚Üí Save as Picture ‚Üí PNG

### Option C: Using Free Online Tool (Recommended)
Use Canva.com or similar:
1. Go to Canva.com (free account)
2. Create custom 1024x1024 design
3. Search for Star of David elements
4. Add clock icon
5. Add text "JC" or Hebrew letter
6. Download as PNG
